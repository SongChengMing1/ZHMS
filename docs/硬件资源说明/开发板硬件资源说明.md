# ESP32-S3-Touch-LCD-1.69 开发板硬件资源说明

## 1. 主控规格

- MCU：ESP32-S3，Xtensa® 32 位 LX7 双核处理器，主频 240 MHz
- 无线：2.4 GHz Wi-Fi (802.11 b/g/n) + Bluetooth® 5 (LE)，板载贴片天线
- 存储：内置 512KB SRAM + 384KB ROM，叠封 8MB PSRAM，外接 16MB Flash

### 2.1 显示 — 1.69" LCD 电容触摸屏

| 项目         | 规格                                    |
| ------------ | --------------------------------------- |
| 屏幕尺寸     | 1.69 英寸，电容触摸                     |
| 分辨率       | 240 × 280 像素，262K 彩色              |
| 显示控制器   | ST7789V2（内置，240×RGB×320 驱动能力） |
| 通信接口     | 四线 SPI（DC/CS/SCL/SDA）              |
| 颜色格式     | RGB565（默认），支持 RGB444 / RGB666    |
| 触摸控制器   | CST816D（自电容触控，I2C 接口）         |
| 触摸设备地址  | 7bit 地址 0x15（写 0x2A / 读 0x2B）    |

> 注：LCD 四角为圆角，边缘部分像素不显示。

### 2.2 六轴 IMU — QMI8658C

- 3 轴陀螺仪 + 3 轴加速度计
- 通信接口：I2C（SCL/SDA 与 RTC/触摸共用 GPIO10/11）

### 2.3 RTC 时钟 — PCF85063A

- 板载 PCF85063A 芯片 + RTC 电池接口（M1.25）
- 通信接口：I2C

### 2.4 电源管理

| 器件       | 功能                             |
| ---------- | -------------------------------- |
| ETA6098    | 锂电池充电管理                   |
| B+ 引脚    | 对外输出电源                     |
| M1.25 接口 | 锂电池接入（充放电）             |
| BAT_ADC    | GPIO1，检测电池电压（V_BAT = V_ADC × 3） |

### 2.5 系统电源控制

| 引脚       | 功能             |
| ---------- | ---------------- |
| KEY_PWR    | 手动按键上电     |
| SYS_EN     | GPIO41，主控上电控制 |
| SYS_OUT    | GPIO40，供电状态反馈 |

| 操作方式       | GPIO41 (SYS_EN) | KEY_PWR | SYS_OUT 状态        |
| -------------- | --------------- | ------- | ------------------- |
| MCU 开机控制   | 高              | 不按    | 高 → 系统供电       |
| MCU 关机控制   | 低              | 不按    | 低 → 系统断电       |
| 手动按键开机   | 任意            | 按下    | 高 → 系统供电       |
| 空闲           | 任意            | 松开    | 由 SYS_EN 决定      |

### 2.6 蜂鸣器

- 引脚：GPIO42，高电平响起(暂不确定有源或无源)

### 2.7 按键

| 按键 | 引脚    | 功能                         |

## 4. LCD 显示控制器 ST7789V2 说明

### 4.1 SPI 通信参数

- 接口：四线 SPI（DC/CS/SCL/SDA，不接 MISO）
- 时钟极性 CPOL = 0（空闲低电平）
- 时钟相位 CPHA = 0（第一个跳变沿采集数据）
- 传输顺序：高位在前，低位在后
- 每时钟周期传输 8 bit 数据

### 4.2 控制时序

| 信号 | 功能说明                                         |
| ---- | ------------------------------------------------ |
| RESX | 复位，上电时拉低，正常工作置高                   |
| CSX  | 片选，低电平使能                                 |
| D/CX | 数据/命令：DC=0 写命令，DC=1 写数据              |
| SDA  | RGB 数据线                                       |
| SCL  | SPI 时钟线                                       |
